# Screenshots

Drop the following PNGs into this folder to populate the README:

| Filename | What it shows | Recommended size |
| --- | --- | --- |
| `screenshot.png` | Main dashboard with score cards + charts | 1200×800 |
| `excel.png` | Excel report (any sheet) | 1200×800 |
| `pdf.png` | PDF report cover or scores page | 1200×800 |
| `demo.gif` | Animated demo of running an analysis | < 5 MB |

The README already references these filenames, so they'll show up
automatically once you commit them.
